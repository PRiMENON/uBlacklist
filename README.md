# uBlacklist

## リスト
* [list.txt](list.txt) [購読](https://raw.githubusercontent.com/primenon/uBlacklist/master/list.txt)

## 使い方

1. uBlacklist を Google Chrome / Mozilla Firefox にインストールします。
1. uBlacklist の設定ページを開きます。
1. 購読のURLをペーストします。
